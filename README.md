# Analise de dados eleitorais do TSE usando Jupyter
________________________________________________________________________________



# Sobre o projeto
análise de dados com a exibição de estatísticas do eleitorado localizado dentro de sua cobertura no Estado de São Paulo (Municipios de Caçapava, Taubaté e Jacareí). Analise contendo gráficos e tabelas de estado civil, escolaridade e faixa etária, além de informações sobre renda e evolução na quantidade de eleitores.


# Executando a aplicação

Depois de baixar o Python (3.9) e clonar o repositório (ou baixá-lo):

Acesse a pasta do projeto por meio do terminal: 

- cd projeto

Instale as dependências:

- pip install -r requirements.txt

# Atualize os dados :
Foi usado como referencia o excelente vídeo do professor Fernando Massanori
	https://www.youtube.com/watch?v=hyEwyiBcQdw
- E utilizado os materiais baixados do sistema do TSE e PDET (ftp://ftp.mtps.gov.br/pdet/microdados/) também gentilmente disponibilizados pelo professor no link :
	​bit.ly/TSEIHUB
- Para que funcione corretamente o projeto é imprescindivel colar na "data" do projeto os arquivos de .csv contendo os dados.  (Arquivos grandes que excederam o limite máximo do github).


Execute a aplicação:

- python app.py

- O serviço do jupyter estará disponível através do link: http://localhost:8888/
